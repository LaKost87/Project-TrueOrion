# Project-TrueOrion
Project-TrueOrion entwickelt sichere, nachweisbare Cloud-Datenlöschung. Ziel ist es, versteckte Kopien, Backups und Snapshots aufzuspüren und durch kryptografische Verfahren endgültig unzugänglich zu machen. Ein Ansatz für echte digitale Vergessenheit und moderne Datenschutz-Compliance.
